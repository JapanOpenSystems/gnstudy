JOPS出力問題
--------------

問題
```
- "J!" "O!" "P!" "S!" をランダムで出力し続けるプラグラムを記述する
- ただし出力末尾が "J!O!P!S!" になったら "JOPS!JOPS!"と出力して処理を終了する

出力サンプル
---
J!P!S!P!S!J!S!J!S!P!P!J!S!S!S!P!O!S!O!J!O!P!S!JOPS!JOPS!
---
```

条件
```
- 用いる言語は自由
- 最も短いコードを作成した人の優勝とする
```

コード提出方法
```
ymorika or hmugish までGistのURL または コード自体を送付してください
```
